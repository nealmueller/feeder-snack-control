macOS utility

One tap. Cat fed.

Feeder Snack Control puts a single feeder action in macOS Control Center. One tap, one snack, with no Dock clutter and no extra installation steps.

Privacy

Control Center first

Built around a single system control for one clear action: send a snack.

Runs quietly

The app can stay in the background instead of living in your Dock or Command-Tab list.

Local by default

Credentials are stored on-device and the action is performed from the Mac that is running the app.
