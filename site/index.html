macOS utility

One tap. Cat fed.

Feeder Snack Control puts a single feeder action in macOS Control Center. One tap, one snack, with no Dock clutter and no extra installation steps.

Download on the App Store

Control Center first

Built around a single system control for one clear action: send a snack.

Runs quietly

The app can stay in the background instead of living in your Dock or Command-Tab list.

Local by default

Credentials are stored on-device and the action is performed from the Mac that is running the app.

Support

Contact neal@nealmueller.com for help with setup or App Store support questions.

Supported hardware

Verified with Feeder-Robot model RF1-8005-0G. Compatibility with other feeder variants has not been verified.

Not currently supported

PETLIBRO One RFID Smart Feeder (PLAF301), PETLIBRO Granary Smart Camera Feeder, PETKIT YumShare Solo Automatic Feeder with Camera, PetSafe Smart Feed Automatic Pet Feeder (2nd Generation), and WOPET Pioneer Plus (FV01 Plus) are not currently supported.
